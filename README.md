# Manufacturing Defect Prediction System

**AI-Powered Quality Control System** for Manufacturing Industry.

## Features
- Real-time defect prediction using Random Forest
- 95.06% Accuracy & 99.27% Recall on Defect Class
- Manual Input + Batch CSV Prediction
- Prediction History (SQLite)
- Interactive Dashboard

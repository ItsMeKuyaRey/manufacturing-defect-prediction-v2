# Manufacturing Defect Prediction System

**AI-Powered Real-time Quality Control System for Manufacturing**

## Project Overview
- **Best Model**: Random Forest (95.06% Accuracy | **99.27% Recall** on Defect Class)
- **Dataset**: 3,240 records, 17 columns
- **Deployment**: Interactive Streamlit Web Application

## Features
- Real-time defect risk prediction
- Manual input and Batch CSV prediction
- Prediction history (SQLite)
- Feature importance analysis
- High recall to minimize costly false negatives
